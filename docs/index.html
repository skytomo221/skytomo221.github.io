人工言語に興味ある？

Discordサーバ「小さな言語交換会♡nipponjo」をクリックして覗いてみよう！
ここではゆるーく人工言語や雑談をすることができます。
つまり、人工言語に詳しくなくても、興味があれば気軽に参加することができます。
また、創作界隈などのどの分野の方でも参加していただけると、
仮想世界を作ろうと試みている人工言語作者にとってとてもいい環境になります。
ぜひ、参加してみてください！
